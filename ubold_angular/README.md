## How did we get started
### Prerequisites
- Node.js (v14 or later)
- npm (v6 or later)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/National-Neurotrauma-Registry/nntr-frontend.git
   cd nntr-frontend/ubold_angular
   ```

2. Install dependencies:
   ```
   npm install
   ```
